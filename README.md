# Companion-tg-bot
Telegram bot for communication support

## Start

Dependies:
1. cherrypy;
2. python-teleram-bot;
3. requests;
4. flask;
5. pyowm==2.10.0;
6. pytelegrambotapi;
7. apiai;
8. SpeechRecognition;
9. pydub;
10. gtts.

**pip3** install cherrypy python-telegram-bot requests flask pyowm==2.10.0 pytelegrambotapi apiai SpeechRecognition pydub gtts

**python3** main.py
